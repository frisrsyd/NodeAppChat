## About NoteAppChat

<p>Realtime chat app mata kuliah Proyek Perangkat Lunak Informatika USK.</p>

## Installation

1. clone the repository

```bash
git clone https://github.com/frisrsyd/NodeAppChat
```

goto folder with this command

```bash
cd NodeAppChat
```

2. Install dependencies

```bash
npm install
```

3. run serve with

```bash
nodemon server.js
```
